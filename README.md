## ✨ Features

- 🎨 **Beautiful UI** - Modern and responsive design with gradient backgrounds
- 🃏 **Product Card** - Display product information in an elegant card layout
- 🔄 **Dynamic Greeting** - Personalized greeting message based on user name
- 🖼️ **Conditional Rendering** - Shows/hides elements based on user input
- 📱 **Fully Responsive** - Works perfectly on all devices
- 🎯 **React-Bootstrap** - Utilizes Bootstrap components for consistent styling
- ✨ **Animations** - Smooth hover effects and fade animations
- 🧩 **Modular Components** - Reusable and well-organized component structure

## 📦 Prerequisites
